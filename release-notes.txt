4.3.1 Release Notes

! Changed behaviour
- Certain utilities like client_drivshell will now require root privileges

! Bug fixes
- Validate stp arguments in ofdpa-grpc instead of passing them on as is

- Fixed issues allowing actors to inject shell commands executed with root
  privileges locally and unauthenticated over the network
