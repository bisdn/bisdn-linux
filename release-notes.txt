BISDN Linux
4.2.0 Release Notes
Date: 2021-10-06

! New in this release
In this release, we add the EdgeCore AS7726-32X as our first Trident3 based 100G switch to the list of supported platforms.

! Bug fixes
- systemd: drop redundant management interface configuration file
