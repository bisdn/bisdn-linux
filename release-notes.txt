BISDN Linux
4.4.0 Release Notes
Date: 2021-12-15

! New in this release
- Edgecore EPS202 / AS4630-54PE (48 x 1GbE RJ45 PoE, 4 x 25GbE SFP28 and 2 x 100GbE QSFP28)
  added as supported platform.
- Updated Linux kernel to 5.15.
- Updated FRR to version 8.1.
- Experimental feature: Support for spanning tree protocol (STP) on bonds.
- Added git package to all images.

! Bug fixes
- basebox: Fixed disabling VLANs on ports currently subscribed to multicast groups
